Create a Game:

https://openchess-api.com/api/new

Add game as a parameter to change the initial setup of the board. Chess, checkers, and none are available options.

Here's all the options

Get Game Board:

https://openchess-api.com/api/game/<game-id>

Add game as a parameter to change the initial setup of the board. Chess, checkers, and none are available options.

Move Piece:

https://openchess-api.com/api/game/<game-id>/<from>-<to>

from and to are coordinates of the form [A-H][1-8] specifying which piece you want to move where. To generate the svg, there must be a piece at the from coordinate.

Set Piece on Board:

https://openchess-api.com/api/game/<game-id>/set/<tile>-<color>-<piece>

Add game as a parameter to change the initial setup of the board. Chess, checkers, and none are available options.

Remove Piece from Board:

https://openchess-api.com/api/game/<game-id>/remove/<tile>

Add game as a parameter to change the initial setup of the board. Chess, checkers, and none are available options.
